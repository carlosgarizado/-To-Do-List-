ToDo List es una aplicación móvil desarrollada con Ionic Angular que permite a los usuarios gestionar sus tareas diarias de forma sencilla e intuitiva.
La app ofrece una interfaz limpia y moderna, con soporte para Android e iOS, y una estructura modular que facilita la ampliación de funcionalidades.

para ejecutar el proyecto solo debes tener instalado node version 20 y ejecutar el comando ionic s
